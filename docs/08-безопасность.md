# Безопасность сайта

Два уровня: защита контента (от копирования) и техническая безопасность (от атак).

---

## Защита контента от копирования

### Защита текстов
- Отключение выделения текста (CSS `user-select: none`)
- Блокировка правой кнопки мыши
- Блокировка горячих клавиш: Ctrl+C, Ctrl+A, Ctrl+U, Ctrl+S
- Частичная блокировка Print Screen (JS)
- Обфускация JS-кода

> ⚠️ 100% защиты не существует. Цель — защититься от массового копирования, не от технически грамотного пользователя.

### Защита изображений
- Водяной знак на всех фото (логотип / URL сайта)
- Изображения как CSS-фон вместо тега `<img>` (сложнее скачать)
- **Hotlink Protection** на хостинге (запрет использования картинок с других сайтов)
- Динамические watermark'и (привязанные к IP посетителя)
- Превью — низкое разрешение; высокое — только после регистрации

### Защита документов
- Ценные материалы — только за формой захвата контакта
- PDF с паролем на редактирование и копирование
- Встраиваемый просмотрщик вместо скачивания (PDF.js)

### Мониторинг кражи контента
- **Copyscape** — проверка текстов на других сайтах
- **Google Alerts** — мониторинг уникальных фраз
- **TinEye / Google Images** — поиск по изображению
- При обнаружении: DMCA-жалоба в Google + письмо нарушителю

## Защита от ботов

- **Cloudflare** (WAF, DDoS защита) — базовый план бесплатно
- **Яндекс SmartCaptcha** / reCAPTCHA v3 — на всех формах
- **Rate limiting** — лимит запросов с одного IP
- **Honeypot-поля** в формах — невидимые поля для ботов

---

## Защита от парсинга

- Цены не хранятся в HTML — подгружаются через защищённый API-запрос
- **Динамический рендеринг** — разный HTML для ботов и людей
- **Fingerprinting** — идентификация парсеров по поведению (без JS, быстрые запросы)
- **Ловушки (honeypot-ссылки)** — невидимые ссылки, переход = блокировка IP
- **Cloudflare Bot Management** — автоблокировка известных парсеров
- Обфускация CSS-классов и структуры DOM

## Техническая безопасность

### Обязательно
| Мера | Инструмент | Стоимость |
|------|-----------|----------|
| SSL-сертификат (HTTPS) | Let's Encrypt | Бесплатно |
| WAF + DDoS защита | Cloudflare Free | Бесплатно |
| Капча на формах | reCAPTCHA v3 | Бесплатно |
| Резервные копии | Хостинг + облако | От 0 ₽ |
| Мониторинг доступности | Uptime Robot | Бесплатно |

### Дополнительно
| Мера | Инструмент | Стоимость |
|------|-----------|----------|
| Продвинутый WAF | Cloudflare Pro | 2 500 ₽/мес |
| Сканирование на вирусы | Sucuri SiteCheck | Бесплатно |
| Двухфакторная авторизация в CMS | Плагин/модуль | Бесплатно |
| Скрытие адреса /admin | Конфигурация | Бесплатно |

### Защита персональных данных (152-ФЗ)
- Данные клиентов — только на серверах в России
- Шифрование базы данных
- Разграничение прав доступа внутри команды
- Регистрация в Роскомнадзоре как оператор ПД
- Политика конфиденциальности на сайте (обязательно)
- Согласие на обработку ПД во всех формах
