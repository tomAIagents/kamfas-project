# Личный кабинет клиента

## Разделы ЛК

| Раздел | URL | Функционал |
|--------|-----|-----------|
| Мои заказы | /account/orders/ | История, статусы, повтор заказа, скачать документы |
| Мои расчёты | /account/calculations/ | Сохранённые расчёты из калькуляторов |
| Мои объекты | /account/projects/ | Комплектация стройки из нескольких категорий |
| Избранное | /account/favorites/ | Сохранённые товары |
| Профиль | /account/profile/ | Данные, смена пароля, настройки уведомлений |
| Документы | /account/documents/ | Счета, акты, накладные, договоры |
| Бонусы и скидки | /account/bonuses/ | Накопленные баллы, уровень лояльности |

## Мои объекты (менеджер проекта)

Ключевая функция для B2B и прорабов: один строительный объект = материалы из нескольких категорий каталога.

```
Объект «Дача Петрова»
├── Фундамент: ФБС-24 × 120 шт, арматура 12мм × 500 кг
├── Стены: Газобетон D500 × 8 поддонов
├── Кровля: Металлочерепица Монтеррей 200 м²
└── Дорожки: Тротуарная плитка Квадрат 60 м²

Итого: 184 500 ₽ | Скачать спецификацию PDF | Добавить всё в корзину
```

## Программа для прорабов (отдельный тип ЛК)

- Спеццены (скидка от базовой наценки)
- Персональный менеджер
- Быстрый повтор предыдущего заказа
- Отсрочка платежа (по договору)
- Расширенная история объектов
- Скачать сводную спецификацию по объекту
