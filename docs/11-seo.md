## Технические требования SEO

- **sitemap.xml** — автогенерация, обновление при добавлении товаров
- **robots.txt** — закрыть корзину, ЛК, поиск от индексации
- **Хлебные крошки** (breadcrumbs) — на всех страницах каталога
- **Canonical URL** — для страниц с фильтрами
- **Open Graph** — мета-теги для соцсетей на каждой странице
- **Schema.org** — разметка товаров (Product, Offer, BreadcrumbList)
- **Скорость** — PageSpeed Score ≥ 80 (мобайл и десктоп)
- **HTTPS** — обязательно
- **Адаптивность** — Mobile-First индексация

## Контент для SEO

- SEO-текст на каждой странице категории (сворачиваемый)
- Блог: минимум 2 статьи в месяц по запросам категорий
- FAQ-блок на категорийных страницах (Schema.org FAQ)
- Отзывы с Schema.org разметкой (рейтинг в выдаче)
