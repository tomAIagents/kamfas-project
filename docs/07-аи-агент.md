# ИИ-агент продажника

## Что это

Встроенный в сайт чат-бот на базе большой языковой модели (Claude API), который работает как менеджер по продажам 24/7: отвечает на вопросы, выявляет потребность, работает с возражениями, квалифицирует лида и передаёт в CRM или живому менеджеру.

## Отличие от обычного чат-бота

| Обычный бот | ИИ-агент |
|-------------|---------|
| Кнопочное меню | Понимает естественную речь |
| Скриптовые ответы | Генерирует ответ по контексту |
| Не работает с возражениями | Умеет убеждать и аргументировать |
| Теряет клиента при нестандарте | Справляется с любым вопросом |

## Сценарии работы

### Квалификация лида
```
Клиент: "Сколько стоит?"
Агент:  "Цена зависит от объёма. Для частного дома или коммерческого объекта?"
Клиент: "Частный дом, нужно 50 м²"
Агент:  "Понял. Укажите телефон — пришлю точный расчёт за 15 минут."
→ Заявка автоматически уходит в CRM
```

### Работа с возражением
```
Клиент: "У вас дорого, у конкурентов дешевле"
Агент:  "Понимаю. Какую цену видели? Часто разница в составе услуги —
         у нас включена доставка и гарантия 2 года. Сравним детально?"
```

### Нерабочее время
```
Клиент в 23:00: "Есть ли товар в наличии?"
Агент: "Да, товар есть. Офис закрыт, но приму заявку. Когда
        перезвонить — в 9:00 или позже?"
```

## База знаний агента

- Полный каталог товаров с ценами
- FAQ (частые вопросы)
- Условия доставки, оплаты, гарантии
- Скрипты работы с возражениями
- Информация о компании и преимуществах
- При незнании ответа → передать менеджеру

## Интеграции агента

```
Сайт (виджет)
    ↓
ИИ-агент
    ├── CRM → создаёт сделку, заполняет карточку
    ├── Telegram → уведомление менеджеру
    ├── WhatsApp → продолжение диалога
    ├── Email → отправка КП / подтверждений
    └── Календарь → запись на звонок/встречу
```

## KPI агента

- % диалогов, завершившихся заявкой
- Количество квалифицированных лидов в сутки
- Среднее время до передачи менеджеру
- Рейтинг диалогов (оценка клиентом)
- Конверсия: с агентом vs без
