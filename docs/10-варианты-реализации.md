## Вариант 1 — Быстрый старт (1С-Битрикс)

```

Хостинг:   VPS 4 CPU / 8 GB RAM
CDN:       Cloudflare Free
```

### Что входит

- Личный кабинет базовый

- Кабинет поставщика

## Вариант 3 — Максимальный (микросервисы + маркетплейс + мобайл)

### Стек
```
Frontend:       Next.js 14 (SSR/SSG) + TypeScript + Tailwind
Mobile:         React Native (iOS + Android)
API Gateway:    Nginx + Kong
Микросервисы:   Node.js + Python FastAPI
  catalog-service    — каталог и поиск
  order-service      — заказы и воронка
  supplier-service   — поставщики и конкуренция цен
  ai-service         — ИИ-агент, рекомендации, голос
  delivery-service   — расчёт доставки и трекинг
  notification-service — все уведомления
  analytics-service  — BI и предиктив
БД:             PostgreSQL + MongoDB + Redis + ClickHouse
Поиск:          Elasticsearch 8
Очереди:        Apache Kafka
Инфраструктура: Yandex Cloud (Kubernetes, auto-scaling)
ИИ:             Claude API + LangChain + RAG + ElevenLabs (голос)
CDN:            Cloudflare Enterprise
Мониторинг:     Grafana + Prometheus + Sentry
```

### Схема маркетплейса поставщиков
| Шаг | Действие | Время |
|-----|---------|-------|
| 1 | Клиент оформляет заказ | — |
| 2 | Параллельный запрос всем поставщикам (10+) | 0–1 сек |
| 3 | Поставщики отвечают: цена + срок + наличие | 1–2 сек |
| 4 | Алгоритм выбирает оптимального (цена + срок + рейтинг) | 0.1 сек |
| 5 | Автоматически размещает заказ через EDI / 1С API | 1–3 сек |
| 6 | Расчёт маршрута и стоимости доставки | 1–2 сек |
| 7 | СБИС генерирует счёт, УПД, акт | 2–5 сек |
| 8 | Клиент получает итог: цена + срок + документы | До 10 сек |
| 9 | Трекинг в реальном времени в ЛК и мобайл | Авто |

## Рекомендация: поэтапное развитие

```
Год 1 (мес. 1–5):   Вариант 2 → запуск, первые продажи
Год 1 (мес. 6–12):  Оптимизация, формирование базы поставщиков
Год 2 (мес. 1–6):   + Мобильное приложение
Год 2 (мес. 7–12):  + Маркетплейс поставщиков
Год 3:              + Голосовой агент, полная автоматизация
```

**Логика:** запускаемся за 5 месяцев → зарабатываем → финансируем следующий этап из прибыли.
