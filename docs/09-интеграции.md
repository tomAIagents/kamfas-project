# Интеграции

## СБИС

| Функция | Что синхронизируется |
|---------|-------------------|
| Цены и остатки | Из СБИС → на сайт в реальном времени |
| Заказы | С сайта → в СБИС автоматически |
| Счета и документы | СБИС генерирует → клиент скачивает в ЛК |
| CRM | Сделки и контакты с сайта → в СБИС CRM |
| ЭДО | Электронный документооборот с B2B клиентами |

## Мессенджеры

### WhatsApp Business API (WABA)
- Виджет на сайте (кнопка «Написать в WhatsApp»)
- Автоответ при первом сообщении
- 7 шаблонов уведомлений:
  1. Заявка принята
  2. Заказ подтверждён + номер
  3. Заказ в пути + трек
  4. Заказ доставлен
  5. Напоминание о брошенной корзине
  6. Запрос отзыва (через 3 дня)
  7. Акция / новинка (рассылка)

### Telegram
- Бот для клиентов: статус заказа, FAQ, заявки 24/7
- Группа менеджеров: уведомления о новых заказах
- Канал компании: новости, акции, новинки

## Email — транзакционные письма

1. Регистрация на сайте
2. Восстановление пароля
3. Заказ оформлен (с составом и суммой)
4. Статус изменился (в пути / доставлен)
5. Счёт на оплату
6. Напоминание о брошенной корзине (через 2 часа)
7. Запрос отзыва (через 3 дня после доставки)
8. Акции и персональные предложения
9. Реактивация (не покупали 3 месяца)

Инструмент: **SendPulse** или **Unisender** (~3 000 — 8 000 ₽/мес)
