# Раздел «Подобрать по виду работ»

## Концепция

Люди ищут не «газобетонный блок D500», а «что нужно для строительства стен».
Раздел `/by-work/` закрывает эту потребность: клиент выбирает вид работ и получает
готовый список всех необходимых материалов из каталога с кнопкой «Добавить всё в корзину».

**Эффект:** рост среднего чека + удобство клиента + SEO-трафик по запросам вида «материалы для фундамента купить».

## Функционал каждой страницы

- **H1:** «Материалы для {вид работ} — купить в Перми»
- Список материалов с фото, ценой, единицей измерения, полем ввода количества
- **Кнопка «Добавить всё в корзину»** — одним кликом
- **Мини-калькулятор:** ввести площадь/объём → система считает количество каждого материала (запас +10%)
- Совет эксперта: текст или видео о правильном выборе
- Блок «Часто берут вместе» (cross-sell)
- SEO-текст внизу (сворачиваемый)
- Ссылки на статьи блога

## Технические требования

- Материалы подтягиваются из каталога автоматически (цена/наличие обновляются везде)
- Администратор редактирует состав набора через CMS
- Количество рассчитывается с запасом 10% (настраивается)
- Страницы индексируются, добавлены в sitemap.xml
- Событие «Добавить всё в корзину» — отдельная цель в Яндекс.Метрике
