## Единицы измерения

| Единица | Применение | Конвертация |
|---------|-----------|-------------|
| м² | Плитка, кровля | м² × кол-во шт/м² = штук |
| м³ | ЖБИ, блоки | м³ × плотность = тонн |
| шт | ЖБИ, блоки, плитка | шт × вес = тонн |
| тонна | ЖБИ (доставка) | тонн ÷ вес изделия = штук |
| лист | Кровля | листов × площадь листа = м² |
| п.м. | Трубы, лотки | п.м. × цена/п.м. = сумма |
| поддон | Блоки, плитка | поддонов × кол-во на поддон = штук |
| бухта | Трубы ПЭ | бухт × длина бухты = п.м. |

## Специфика карточки по категориям

| Категория | Уникальные поля | Особые кнопки |
|-----------|----------------|---------------|
| ЖБИ | Марка бетона, ГОСТ, серия, масса, схема армирования | Скачать паспорт, Запросить нестандарт |
| Кровля | Толщина металла, покрытие, RAL-цвет | Рассчитать кровлю, Инструкция |
| Стены | Марка, плотность, теплопроводность, размер | Рассчитать стены, Характеристики |
| Плитка | Толщина, вес м², рисунок укладки, цвет | Рассчитать плитку, Образец, Укладка |
| Коммуникации (газ) | Тип изделия, ГОСТ, класс нагрузки | ⚠️ Монтаж только лицензированными |

| Кольца колодезные | Трубы ПВХ, геотекстиль, щебень, люки |
| Трубы водопроводные | Фитинги, запорная арматура, утеплитель |
