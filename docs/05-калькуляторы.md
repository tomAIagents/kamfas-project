# Калькуляторы материалов

Единая система расчётов на странице /calculators/ — хаб с выбором типа.
Все калькуляторы работают в реальном времени, цены из актуального каталога,
результат сохраняется в ЛК → «Мои расчёты».

## Калькулятор тротуарной плитки

**Входные данные:**
- Площадь укладки (м²) или размеры (длина × ширина)
- Тип плитки (из каталога с фото)
- Цвет
- Рисунок укладки (прямой / диагональ / ёлочка / плетёнка)
- Наличие бордюра (да/нет, длина периметра)

**Результат:**
- Количество плитки (м² + штук + поддонов)
- Количество бордюрного камня
- Количество песка и щебня для основания
- Итоговая стоимость + стоимость доставки
- PDF-смета | Добавить в корзину | Сохранить в ЛК

## Калькулятор кровли

**Входные данные:**
- Форма крыши (двускатная / вальмовая / односкатная)
- Размеры (длина конька, ширина пролёта)
- Угол наклона
- Тип кровельного материала (из каталога)

**Результат:**
- Площадь кровли (м²)
- Количество листов / рулонов
- Доборные элементы (конёк, ендова, карниз)
- Крепёж (саморезы по типу материала)
- Утеплитель и пароизоляция
- Итоговая стоимость

## Калькулятор стен

**Входные данные:**
- Площадь стен (м²) или размеры здания
- Количество и размеры проёмов (окна, двери)
- Тип кладочного материала (из каталога)
- Толщина кладки

**Результат:**
- Количество блоков / кирпичей (штук + поддонов)
- Объём кладочного раствора
- Кладочная сетка (п.м.)
- Итоговая стоимость

## Калькулятор ЖБИ

**Входные данные:**
- Тип объекта (фундамент / перекрытие / колодец)
- Размеры (периметр, длина, глубина)

**Результат:**
- Количество изделий (ФБС, плиты ПК, кольца)
- Тоннаж (для расчёта доставки манипулятором)
- Итоговая стоимость

## Технические требования

- Мгновенный пересчёт при изменении параметров (без кнопки)
- Цены из каталога в реальном времени
- Запас материала +10% (настраивается в CMS)
- Сохранение результата в «Мои расчёты» ЛК
- Добавление результата в «Мой объект»
- Скачать PDF-смету
- На мобильном: числовая клавиатура (inputmode="numeric")
