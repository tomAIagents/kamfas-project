# Торговля стройматериалами — КамФас

## О проекте

Разработка полноценной торговой платформы для компании **КамФас** (Пермский край).

Сайт заменяет текущий [kamfas.ru](https://kamfas.ru) и функционирует как единая платформа:
интернет-магазин строительных материалов с личным кабинетом, ИИ-агентом продажника,
системой подбора материалов по виду работ и автоматической интеграцией с поставщиками.

## Ключевые параметры проекта

- **Текущий сайт:** [kamfas.ru](https://kamfas.ru)
- **Тип:** B2C + B2B интернет-магазин строительных материалов
- **Производство:** ЖБИ, тротуарная плитка (собственное)
- **Партнёрский товар:** Кровля, Стены, Утеплители, Листовые, Пиломатериалы, Сухие смеси, Металлопрокат, Крепёж, Фасад, Коммуникации
- **Регион:** Пермский край

## Быстрый старт по разделам

### Каталог (12 разделов, 66 подкатегорий)

```
ЖБИ → Кровля → Стены → Плитка → Утеплители → Листовые →
Пиломатериалы → Сухие смеси → Металлопрокат → Крепёж → Фасад → Коммуникации
```
